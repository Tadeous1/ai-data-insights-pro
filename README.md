# AI Data Insights Pro

This repository contains the AI Data Insights Pro application and its setup files.

## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/Tadeous1/ai-data-insights-pro.git
   ```
2. **Install dependencies**
   ```bash
   npm install
   ```
3. **Run the application**
   ```bash
   npm start
   ```
